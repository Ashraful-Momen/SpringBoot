# Spring Boot Startup-এর Real Story

```text
main()

   │

   ▼

SpringApplication.run()

   │

   ▼

Create ApplicationContext

   │

   ▼

Component Scan

   │

   ▼

Find

@Component
@Service
@Repository
@Configuration

   │

   ▼

Create Objects

   │

   ▼

Store Objects

   │

   ▼

Dependency Injection

   │

   ▼

Application Ready
```

এখানে সবচেয়ে গুরুত্বপূর্ণ জিনিস হলো:

```text
@Configuration
```

বলছে:

"এই class এ Bean তৈরির factory method আছে"

আর

```text
@Bean
```

বলছে:

"এই method call করে object বানাও এবং container-এ রাখো"

এবং

```text
@RequiredArgsConstructor
```

বলছে:

"Constructor আমি generate করে দিচ্ছি"

কিন্তু object create করে **Spring Container**, Lombok না। Lombok শুধু constructor code লিখে দেয়।
