# Step 3: Publish Event

## CaseService.java

```java
@Service
public class CaseService {

    @Autowired
    private CaseRepository repository;

    @Autowired
    private ApplicationEventPublisher
            publisher;

    public SupportCase createCase(

            CaseRequest request

    ){

        SupportCase supportCase =
                new SupportCase();

        supportCase.setTitle(
                request.getTitle()
        );

        supportCase.setDescription(
                request.getDescription()
        );

        SupportCase savedCase =
                repository.save(
                        supportCase
                );

        publisher.publishEvent(

            new CaseCreatedEvent(

                savedCase.getId(),

                savedCase.getTitle()

            )
        );

        return savedCase;
    }
}
```
