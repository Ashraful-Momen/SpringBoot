# Step 4

# Service Layer

```java
@Service
@RequiredArgsConstructor
public class UserService {

    private final UserRepository repository;

    public User create(User user){

        return repository.save(user);
    }

    public List<User> all(){

        return repository.findAll();
    }
}
```

# @RequiredArgsConstructor

Lombok creates:

```java

   this.repository = repository;
}
```

Automatically.

Why?

Because Spring uses:

```text

```

Flow

```text

 ↓
```

# Controller

# Most Important Thing To Remember

Spring Boot Core Heart:

```text
@SpringBootApplication

        ↓

Component Scan

        ↓

Find
@Component
@Service
@Repository
@Configuration

        ↓

Create Beans

        ↓

Store In IoC Container

        ↓

Dependency Injection

        ↓

Application Ready
```

যখন তুমি এই Flow চোখ বন্ধ করে explain করতে পারবে—`@SpringBootApplication → Component Scan → Bean Creation → IoC Container → Dependency Injection → Controller → Service → Repository`—তখন Spring Boot Core Architecture-এর foundation তোমার পরিষ্কার হয়ে যাবে। এরপর JPA, Security, Transaction, Cache, Event, Redis—সবই এই foundation-এর উপর বসে।
