# DispatcherServlet কী Spring Boot এ?

Spring MVC-এর **Heart** হলো `DispatcherServlet`।

সহজ ভাষায়:

> DispatcherServlet হলো Front Controller, যা সব HTTP Request প্রথমে Receive করে এবং সঠিক Controller-এর কাছে পাঠায়।

Architecture:

```txt
Browser

   │

   ▼

DispatcherServlet

   │

   ▼

Controller

   │

   ▼

Service

   │

   ▼

Repository

   │

   ▼

Database
```

# Example

## Controller

```java
@RestController
@RequestMapping("/students")
public class StudentController {

    @GetMapping("/{id}")
    public String getStudent(
            @PathVariable Long id
    ){

        return "Student " + id;
    }
}
```

User Request:

```http
GET /students/1
```

DispatcherServlet:

```txt
Find Controller

Find Method

Execute Method
```

Then:

```java
getStudent(1L)
```

Run হবে।

# Interview Answer (Short)

**What is DispatcherServlet?**

> DispatcherServlet is the Front Controller of Spring MVC. It receives all incoming HTTP requests, finds the appropriate controller using HandlerMapping, executes the controller method, and returns the response back to the client.
