# What is JPA?

JPA = Java Persistence API

এটা Specification।

নিজে কোনো Code Execute করে না।

শুধু Rules Define করে।

Example:

```java
@Entity
@Id
@Table
```

এসব JPA Specification।

---

# What is Hibernate?

Hibernate = JPA Implementation

Hibernate JPA Rules Follow করে।

Hibernate এর কাজ:

```txt
Java Object
      ↓
SQL Query
      ↓
Database
```

Convert করা।

# Entity

Entity = Database Table এর Java Representation

---

# @Entity

```java
@Entity
public class Student {

}
```

মানে:

```txt
Student Class
      ↓
students Table
```

Hibernate বুঝে যায় এটা Database Table।

---

# @Table

Custom Table Name Define করে।

```java
@Entity
@Table(name = "students")
public class Student {

}
```

Database:

```sql
students
```

Table Create হবে।

---

# Without @Table

```java
@Entity
public class Student {

}
```

Hibernate Automatically:

```sql
student
```

Table Name বানাবে।

---

# @Id

Primary Key Define করে।

```java
@Id
private Long id;
```

Database:

```sql
id BIGINT PRIMARY KEY
```

---

# @GeneratedValue

Auto Increment ID।

```java
@Id
@GeneratedValue(
    strategy = GenerationType.IDENTITY
)
private Long id;
```

Database:

```txt
1
2
3
4
5
```

Automatically Generate হবে।

---

# Generation Types

```java
GenerationType.IDENTITY
```

Database Auto Increment।

---

```java
GenerationType.SEQUENCE
```

PostgreSQL Sequence Use করে।

---

```java
GenerationType.AUTO
```

Hibernate Decide করবে।

---

# @Column

Column Customize করতে।

```java
@Column(
    name = "student_name",
    nullable = false,
    unique = true,
    length = 100
)
private String name;
```

Database:

```sql
student_name VARCHAR(100)
NOT NULL
UNIQUE
```

---

# Example Entity

```java
@Entity
@Table(name = "students")
public class Student {

    @Id
    @GeneratedValue(
        strategy = GenerationType.IDENTITY
    )
    private Long id;

    @Column(nullable = false)
    private String name;

    @Column(unique = true)
    private String email;

}
```

---

# @Enumerated

Enum Database এ Store করতে।

---

# Without Enum

```java
private String status;
```

Risky।

---

# Create Enum

```java
public enum Status {

    ACTIVE,
    INACTIVE,
    BLOCKED

}
```

---

# Use Enum

```java
@Enumerated(
    EnumType.STRING
)
private Status status;
```

Database:

```txt
ACTIVE
INACTIVE
BLOCKED
```

Store হবে।

---

# Why STRING?

❌

```java
EnumType.ORDINAL
```

Store করবে:

```txt
0
1
2
```

Dangerous।

---

✔

```java
EnumType.STRING
```

Always Use।

# Lifecycle Diagram

```txt
NEW
 ↓

 ↓
DETACHED
 ↓

```

---

# NEW State

Object Create হয়েছে।

Database এ নেই।

```java
Student student =
    new Student();
```

---

# MANAGED State

Entity Persistence Context এর ভিতরে আছে।

```java
studentRepository.save(student);
```

Hibernate Track করছে।

---

# DETACHED State

Persistence Context এর বাইরে।

Hibernate আর Track করছে না।

```java
entityManager.detach(student);
```

---

# REMOVED State

Delete হওয়ার জন্য Marked।

```java
entityManager.remove(student);
```

# Persistence Context

Hibernate এর Memory Area।

---

# Concept

```txt
Database
      ↑
Persistence Context
      ↑
Hibernate
```

সব Entity আগে এখানে Load হয়।

---

# Example

```java
Student student =
entityManager.find(
    Student.class,
    1L
);
```

Database থেকে Load হয়ে:

```txt
Persistence Context
```

এ Store হবে।

---

# Why Important?

কারণ Hibernate একই Object বারবার Database থেকে আনে না।

---

# First Level Cache

Persistence Context = First Level Cache

---

# Example

```java
Student s1 =
entityManager.find(
Student.class,
1L
);

Student s2 =
entityManager.find(
Student.class,
1L
);
```

Database Hit কয়বার?

```txt
Only 1 Time
```

কারণ দ্বিতীয়বার Cache থেকে দিবে।

---

# Flow

```txt
find()
   ↓
Persistence Context Check
   ↓
Found?
   ↓
YES
   ↓
Return Cache
```

# Dirty Checking

Hibernate এর Magic Feature।

---

# Example

```java
@Transactional
public void updateStudent(){

    Student student =
    repository.findById(1L)
              .orElseThrow();

    student.setName("Ashraful");

}
```

Notice:

```java
repository.save(...)
```

নেই।

# @Transactional

---

# Purpose

```txt
Atomic Operation
```

Ensure করে।

---

# Example

```java

}
```

# If Error Occurs

```java
@Transactional
public void transferMoney(){

    updateBalance();

    int x = 10 / 0;

    saveTransaction();
}
```

```txt
ROLLBACK
```

# Transaction Flow

```txt

  ↓

  ↓
```
