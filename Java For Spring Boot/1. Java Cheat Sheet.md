# Java Cheat Sheet for Spring Boot Developers

এটা এমনভাবে সাজানো হয়েছে যাতে Spring Boot backend development করার জন্য প্রয়োজনীয় Java concepts দ্রুত revise করতে পারেন।

---

# 1. Class & Object

```java
class User {
    String name;

    void printName() {
        System.out.println(name);
    }
}

public class Main {
    public static void main(String[] args) {
        User u = new User();
        u.name = "Ashraful";
        u.printName();
    }
}
```

---

# 2. Constructor

```java
class User {
    String name;

    User(String name) {
        this.name = name;
    }
}
```

---

# 3. Encapsulation (IMPORTANT)

```java
class User {
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
}
```

Spring Boot entities/DTO তে অনেক use হবে।

---

# 4. Inheritance

```java
class Animal {
    void sound() {
        System.out.println("Sound");
    }
}

class Dog extends Animal {
}
```

---

# 5. Interface (VERY IMPORTANT)

```java
interface Payment {
    void pay();
}

class BkashPayment implements Payment {
    public void pay() {
        System.out.println("Paid");
    }
}
```

Spring heavily interface-based।

# 10. Loop

```java

```

---

# 11. Stream API (VERY IMPORTANT)

## Filter

```java
List<String> result = names.stream()
    .filter(n -> n.startsWith("A"))
    .toList();
```

## Map

```java
List<Integer> nums = List.of(1,2,3);

List<Integer> doubled = nums.stream()
    .map(n -> n * 2)
    .toList();
```
