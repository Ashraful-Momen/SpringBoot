আপনি Java 21 use করছেন, তাই আমি modern Java style এ একদম simple real-world example দিয়ে বুঝাচ্ছি।

# 1. List — Ordered Collection

ধরেন classroom এর student list।

```java
List<String> students = new ArrayList<>();

students.add("Ash");
students.add("Rahim");

System.out.println(students);
```

Output:

```txt
[Ash, Rahim]
```

## Characteristics

* duplicate allowed
* order maintain করে
* index আছে

```java
System.out.println(students.get(0));
```

Output:

```txt
Ash
```

---

# 2. List.of() — Immutable List

Java modern shortcut।

```java
List<String> names = List.of("Ash", "Sakib", "Hasan");

System.out.println(names);
```

এখানে:

* add/remove করা যাবে না
* fixed list

```java
names.add("New"); // ERROR
```

Spring Boot DTO test data তে অনেক use হয়।

---

# 3. Set — Unique Collection

Duplicate remove করতে use হয়।

```java
Set<String> fruits = new HashSet<>();

fruits.add("Apple");
fruits.add("Apple");
fruits.add("Banana");

System.out.println(fruits);
```

Output:

```txt
[Apple, Banana]
```

## Characteristics

* duplicate allowed না
* mostly unordered

Use case:

* unique emails
* unique roles
* permissions

---

# 4. Map — Key Value Pair

Dictionary এর মতো।

```java
Map<String, Integer> products = new HashMap<>();

products.put("Apple", 120);
products.put("Mango", 80);

System.out.println(products);
```

Output:

```txt
{Apple=120, Mango=80}
```

Access:

```java
System.out.println(products.get("Apple"));
```

Output:

```txt
120
```

Use case:

* userId → user
* token → user
* config storage

---

# 5. Arrow Function / Lambda

পুরাতন style:

```java
for(String name : names){
    System.out.println(name);
}
```

Modern Java:

```java
names.forEach(name -> {
    System.out.println(name);
});
```

এখানে:

```java
name -> {}
```

এইটাই lambda / arrow function।

মানে:

> “এই variable নিয়ে এই কাজ করো”

Short form:

```java
names.forEach(System.out::println);
```

---

# 6. Streams — Data Pipeline

Streams = collection এর উপর modern processing system।

ধরেন:

```java
List<Integer> nums = List.of(1,2,3,4,5);
```

---

## Filter

Even number বের করি।

```java
List<Integer> even = nums.stream()
        .filter(n -> n % 2 == 0)
        .toList();

System.out.println(even);
```

Output:

```txt
[2, 4]
```

---

## Map

সব number double করি।

```java
List<Integer> doubled = nums.stream()
        .map(n -> n * 2)
        .toList();

System.out.println(doubled);
```

Output:

```txt
[2, 4, 6, 8, 10]
```

---

## Real Spring Boot Example

```java
List<User> activeUsers = users.stream()
        .filter(user -> user.isActive())
        .toList();
```

মানে:

> active user গুলো বের করো।

# 7. Generic — Dynamic Type

Generic মানে:

> type reusable করা

Without generic:

```java
Object data;
```

Problem:

* type safe না

---

With generic:

```java
class Box<T> {

    T value;

    void set(T value){
        this.value = value;
    }

    T get(){
        return value;
    }
}
```

Use:

```java
Box<String> box = new Box<>();

box.set("Hello");

System.out.println(box.get());
```

এখানে:

```txt
T = String
```

---

## Why Important?

Spring internally heavily generic use করে।

Example:

```java
ResponseEntity<UserDTO>
List<User>
Optional<User>
```

# 8. Optional — Null Safety

Java তে null অনেক dangerous।

পুরাতন style:

```java
User user = null;

System.out.println(user.getName());
```

Boom 💥

```txt
NullPointerException
```

---

Modern Java:

```java
Optional<String> name = Optional.of("Ash");
```

Get value:

```java
System.out.println(name.get());
```

---

## Empty Optional

```java
Optional<String> name = Optional.empty();
```

Safe check:

```java
if(name.isPresent()){
    System.out.println(name.get());
}
```

---

## Best Modern Style

```java
name.ifPresent(System.out::println);
```

---

## Real Spring Boot Example

```java
Optional<User> user = userRepository.findById(id);
```

মানে:

> user থাকতে পারে, নাও থাকতে পারে।

# 9. Multithreading — Multiple Tasks Together

Normally:

```java
task1
task2
task3
```

একটার পর একটা।

---

Multithreading:

```java
task1 & task2 together
```

---

## Simple Thread Example

```java
Runnable task = () -> {
    System.out.println("Task Running...");
};

Thread thread = new Thread(task);

thread.start();
```

Output:

```txt
Task Running...
```

---

# Real Example

ধরেন:

* email send
* sms send
* notification send

এগুলো background এ run করবে।

---

# 10. CompletableFuture (Modern Multithreading)

Java 21 এ খুব important।

```java
CompletableFuture.runAsync(() -> {
    System.out.println("Background Task");
});
```

Main thread block হবে না।

---

## Example

```java
System.out.println("Start");

CompletableFuture.runAsync(() -> {
    System.out.println("Heavy Task");
});

System.out.println("End");
```

Possible output:

```txt
Start
End
Heavy Task
```

কারণ async background এ run হচ্ছে।

---

# 11. Combined Real Example

```java
List<String> users = List.of(
        "Ash",
        "Rahim",
        "Karim"
);

users.stream()
        .filter(name -> name.startsWith("A"))
        .map(String::toUpperCase)
        .forEach(System.out::println);
```

Step by step:

* stream শুরু
* A দিয়ে শুরু filter
* uppercase convert
* print

Output:

```txt
ASH
```
