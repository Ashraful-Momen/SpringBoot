# 1. Generic — “Dynamic Type Template”

ধরেন JavaScript এ:

```javascript id="b8otm9"
function box(value){
   return value;
}
```

এখানে:

* string দিলেও কাজ করে
* number দিলেও কাজ করে

কারণ JS dynamic।

---

কিন্তু Java strict typed।

তাই Java বলে:

> “আগে type বলে দাও”

---

# Generic ছাড়া Problem

```java id="lr5m59"
class Box {

    Object value;

    void set(Object value){
        this.value = value;
    }

    Object get(){
        return value;
    }
}
```

Use:

```java id="5i5wtm"
Box box = new Box();

box.set("Hello");
```

Problem:

```java id="nbqj2z"
String name = (String) box.get();
```

বারবার casting করতে হবে 😑

---

# Generic দিয়ে Solution

```java id="2v5scs"
class Box<T> {

    T value;

    void set(T value){
        this.value = value;
    }

    T get(){
        return value;
    }
}
```

এখন:

```java id="b0wkqn"
Box<String> box = new Box<>();

box.set("Ash");

String name = box.get();
```

No casting 😎

---

# T মানে কী?

```txt id="j1of3o"
T = Type
```

মানে:

> পরে type দিবো।

# 2. Optional — “Null Safe Box”

Optional মানে:

> value থাকতে পারে, নাও থাকতে পারে।

---

# Problem Without Optional

```java id="jyhz91"
User user = null;

System.out.println(user.getName());
```

Boom 💥

```txt id="9cztn4"
NullPointerException
```

---

# Optional দিয়ে

```java id="j3mz6h"

        Optional.of("Ash");
```

এখানে:

```txt id="pgx3mk"

```

# Empty Optional


মানে:


# Safe Use

# 4. Stream = JavaScript Array Methods

# 6. Python List Comprehension Equivalent

# 7. Yield in Java

Java 21 modern switch:

```java id="pvjz0o"
String result = switch(day){

    case "MONDAY" -> {
        yield "Work";
    }

    case "FRIDAY" -> {
        yield "Weekend Mood";
    }

    default -> {
        yield "Normal";
    }
};
```
