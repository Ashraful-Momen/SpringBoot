package com.unisoftai.lms;

import org.springframework.web.bind.annotation.RestController;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;


@RestController
public class HelloController {
    @GetMapping("/hello")
    public String sayHello(@RequestParam String param) {
        System.out.println(param);
        return "Hello This is the LMS Project ";
    }
    
}

Done! I've added spring-boot-devtools to your pom.xml.

Now restart your project:


>>> ./mvnw spring-boot:run
How it works:

After restarting, any code change you save will automatically trigger a restart (usually within 1-2 seconds)
Just refresh the browser to see the changes
No need to manually stop and start the server
Note: The first restart after adding devtools will take a moment to download the dependency. After that, hot-reloads will be fast.
