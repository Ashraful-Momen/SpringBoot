# Step 1 — Docker দিয়ে PostgreSQL + Redis + pgAdmin চালু করো

`docker-compose.yml` (project root এ already আছে):

```yaml
services:
  postgres:
    image: postgres:16
    container_name: lms_postgres
    environment:
      POSTGRES_DB: lms_db
      POSTGRES_USER: lms_user
      POSTGRES_PASSWORD: lms_password
    ports:
      - "5433:5432"
    volumes:
      - lms_postgres_data:/var/lib/postgresql/data
    restart: unless-stopped

  redis:
    image: redis:7-alpine
    container_name: lms_redis
    ports:
      - "6380:6379"
    volumes:
      - lms_redis_data:/data
    restart: unless-stopped

  pgadmin:
    image: dpage/pgadmin4:latest
    container_name: lms_pgadmin
    environment:
      PGADMIN_DEFAULT_EMAIL: admin@lms.com
      PGADMIN_DEFAULT_PASSWORD: admin123
    ports:
      - "5051:80"
    depends_on:
      - postgres
    restart: unless-stopped

volumes:
  lms_postgres_data:
  lms_redis_data:
```

Start করো:

```bash
docker compose up -d
```

| Service | Container | Port | Access |
|---------|-----------|------|--------|
| PostgreSQL | lms_postgres | **5433** | `jdbc:postgresql://localhost:5433/lms_db` |
| Redis | lms_redis | **6380** | `localhost:6380` |
| pgAdmin | lms_pgadmin | **5051** | http://localhost:5051 |

pgAdmin Login: `admin@lms.com` / `admin123`
PostgreSQL Connection (pgAdmin এ): Host: `lms_postgres`, Port: `5432`, User: `lms_user`, Password: `lms_password`

# Step 5 — Create Controller

## StudentController.java

```java
package com.unisoftai.lms.controller;

import com.unisoftai.lms.model.Student;
import com.unisoftai.lms.repository.StudentRepository;

import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.*;

import java.util.List;

@RestController
@RequestMapping("/students")
public class StudentController {

    @Autowired
    private StudentRepository studentRepository;

    // CREATE
    @PostMapping
    public Student createStudent(@RequestBody Student student) {
        return studentRepository.save(student);
    }

    // READ ALL
    @GetMapping
    public List<Student> getAllStudents() {
        return studentRepository.findAll();
    }

    // READ SINGLE
    @GetMapping("/{id}")
    public Student getStudent(@PathVariable Long id) {
        return studentRepository.findById(id).orElse(null);
    }

    // UPDATE
    @PutMapping("/{id}")
    public Student updateStudent(
            @PathVariable Long id,
            @RequestBody Student studentDetails) {

        Student student = studentRepository.findById(id).orElse(null);

        if (student != null) {

            student.setName(studentDetails.getName());
            student.setEmail(studentDetails.getEmail());

            return studentRepository.save(student);
        }

        return null;
    }

    // DELETE
    @DeleteMapping("/{id}")
    public String deleteStudent(@PathVariable Long id) {

        studentRepository.deleteById(id);

        return "Student Deleted Successfully";
    }
}
```

# Important Annotations

| Annotation        | কাজ                     |
| ----------------- | ----------------------- |
| `@Entity`         | Database table তৈরি করে |
| `@Table`          | Table name specify করে  |
| `@Id`             | Primary key             |
| `@GeneratedValue` | Auto increment ID       |
| `@RestController` | REST API controller     |
| `@RequestMapping` | Base URL set করে        |
| `@RequestBody`    | JSON body receive করে   |
| `@PathVariable`   | URL id receive করে      |
| `@Autowired`      | Dependency inject করে   |
