# HelloController.java Code

```java id="oqz4uu"
package com.unisoftai.lms;

import org.springframework.web.bind.annotation.RestController;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;

@RestController
public class HelloController {

    @GetMapping("/hello")
    public String sayHello(@RequestParam String param) {

        System.out.println(param);

        return "Hello This is the LMS Project ";

    }

}
```

---

# Code Explanation

| Annotation / Code       | কাজ                                |
| ----------------------- | ---------------------------------- |
| `@RestController`       | Class কে REST API Controller বানায় |
| `@GetMapping("/hello")` | GET API Route তৈরি করে             |
| `@RequestParam`         | URL parameter receive করে          |
| `System.out.println()`  | Console এ data print করে           |
| `return`                | Response browser এ পাঠায়           |

# Enable Live Reload / Hot Reload

যদি code save করার সাথে সাথে automatic restart এবং live update দেখতে চাও, তাহলে `pom.xml` এ নিচের dependency add করো।

---

# Add Dependencies in pom.xml

```xml id="u08cx9"
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-test</artifactId>
    <scope>test</scope>
</dependency>

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-devtools</artifactId>
    <scope>runtime</scope>
</dependency>
```

---

# Restart the Project

Dependency add করার পরে আবার run করো:

```bash id="pclcjq"
./mvnw spring-boot:run
```

# DevTools কী কাজ করে?

Spring Boot DevTools ব্যবহার করলে:

* Code save করলে automatic restart হবে
* Browser refresh দিলেই updated result দেখাবে
* বারবার server stop/start করতে হবে না
* Development অনেক fast হয়ে যায়
