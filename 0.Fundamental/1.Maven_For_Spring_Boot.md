## Maven কী Spring Boot এ?

Apache Maven হলো Java project manage করার একটি **Build Automation Tool**।
Spring Boot project এ Maven ব্যবহার করা হয় mainly:

* Dependency manage করতে
* Project build করতে
* Jar/War file generate করতে
* Project run করতে
* Library version manage করতে

সহজভাবে বললে:

> Maven হলো Spring Boot project এর “ম্যানেজার” যেটা project এর সব library, build process, package management automatically handle করে।

---

# Real Life Example

ধরো তুমি একটি Spring Boot project বানাচ্ছো।

তোমার দরকার:

* MySQL Driver
* Spring Web
* Spring Security
* JWT Library
* Lombok

এগুলো manually download করলে অনেক ঝামেলা।

Maven শুধু `pom.xml` এ লিখলেই সব internet থেকে download করে manage করে দেয়।

---

# Maven এর Main কাজগুলো

## 1. Dependency Management

যে library দরকার Maven automatically download করে।

Example:

```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-web</artifactId>
</dependency>
```

এটা লিখলে Maven:

* Spring MVC
* Tomcat
* Jackson
* Logging

সব automatically download করবে।

---

# 2. Build Automation

Maven automatically:

* compile
* test
* package
* build

করতে পারে।

---

# 3. JAR/WAR Generate

Command:

```bash
mvn package
```

এরপর:

```bash
target/project-name.jar
```

generate হবে।

Run:

```bash
java -jar project-name.jar
```

---

# 4. Dependency Version Control

যদি different library version conflict হয় Maven সেটা handle করে।

এটাকে বলে:

* Dependency Resolution
* Transitive Dependency Management

---

# `pom.xml` কী?

এটা Maven এর main configuration file।

Full meaning:

```txt
POM = Project Object Model
```

এখানে define করা হয়:

* project name
* version
* dependencies
* plugins
* java version
* build settings

---

# Simple `pom.xml` Example

```xml
<project>
    
    <modelVersion>4.0.0</modelVersion>

    <groupId>com.example</groupId>
    <artifactId>demo</artifactId>
    <version>1.0.0</version>

    <dependencies>

        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-web</artifactId>
        </dependency>

    </dependencies>

</project>
```

# Spring Boot + Maven Relationship

Spring Boot এবং Maven খুব tightly integrated।

Spring Boot project create করলে default হিসেবে Maven use হয়।

Example dependencies:

```xml
spring-boot-starter-web
spring-boot-starter-data-jpa
spring-boot-starter-security
```

এগুলো Maven automatically manage করে।

# Maven vs Gradle

| Feature             | Maven     | Gradle        |
| ------------------- | --------- | ------------- |
| Config              | XML       | Groovy/Kotlin |
| Easy to Learn       | Yes       | Medium        |
| Performance         | Medium    | Faster        |
| Spring Boot Support | Excellent | Excellent     |
| Beginner Friendly   | Very High | Medium        |

# কেন Spring Boot এ Maven এত Important?

কারণ:

* Easy build process
* Standard project structure
* CI/CD friendly
* Docker friendly
* Jenkins/GitHub Actions compatible
