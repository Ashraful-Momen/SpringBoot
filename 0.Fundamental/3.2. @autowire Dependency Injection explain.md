# `@Autowired` কী?

Spring Framework এ `@Autowired` ব্যবহার করা হয় automatic object injection এর জন্য।

সহজ ভাষায়:

> Spring automatically object তৈরি করে দেয় এবং inject করে দেয়।

---

# প্রথমে Problem বুঝি

ধরো তোমার একটা class আছে:

```java id="5naxw5"
public class MyService {

    public void sayHello() {
        System.out.println("Hello");
    }

}
```

এখন অন্য class এ এটা use করতে চাইলে normally:

```java id="11ejxw"
public class MyController {

    MyService myService = new MyService();

}
```

এখানে তুমি manually object বানাচ্ছো।

---

# Spring কী করে?

Spring নিজেই object manage করে।

মানে:

```txt id="v7jlwm"
Spring Container
      ↓
Object Create
      ↓
Object Inject
```

---

# `@Autowired` Example

---

# Step 1 — Service Class

```java id="4eqkyl"
import org.springframework.stereotype.Service;

@Service
public class MyService {

    public String getMessage() {
        return "Hello From Service";
    }

}
```

---

# `@Service` কী?

```java id="jlwmm1"
@Service
```

মানে:

> Spring এই class এর object তৈরি করবে।

---

# Step 2 — Controller

```java id="jlwmm2"
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class MyController {

    @Autowired
    private MyService myService;

    @GetMapping("/hello")
    public String hello() {

        return myService.getMessage();

    }

}
```

---

# এখানে কী হলো?

এই line:

```java id="jlwmm3"
@Autowired
private MyService myService;
```

মানে:

```txt id="jlwmm4"
Spring → MyService object তৈরি করবে
        → myService এর ভিতরে inject করবে
```

---

# Internally Spring কী করে?

Spring internally:

```java id="jlwmm5"

```

এটা automatically করে দেয়।

# Types of Injection

| Type                  | Recommended   |
| --------------------- | ------------- |
| Field Injection       | Beginner      |
| Constructor Injection | Best Practice |
| Setter Injection      | Sometimes     |

---

# Field Injection (তুমি এখন এটা use করছো)

```java id="jlwmml"
@Autowired
private MyService myService;
```

---

# Constructor Injection (Industry Standard)

```java id="jlwmmm"
@RestController
public class MyController {

    private final MyService myService;

    public MyController(MyService myService) {
        this.myService = myService;
    }

}
```

Modern Spring এ এটা বেশি recommended।

---

# Why Constructor Injection Better?

* immutable
* easier testing
* cleaner design
* better for large projects
