## Repository কী Spring Boot এ?

Spring Data JPA এ `Repository` হলো database এর সাথে communication করার layer।

সহজ ভাষায়:

> Repository database থেকে data আনা, save করা, update করা, delete করার কাজ করে।

মানে:

```txt id="nyrc7j"
Java Code  ⇄  Repository  ⇄  Database
```

---

# Real Life Example

ধরো তোমার কাছে একটা Student table আছে।

তুমি চাইতেছো:

* student save করতে
* সব student দেখতে
* student update করতে
* delete করতে

এই database operations Repository handle করে।

---

# Repository ছাড়া কী হতো?

তোমাকে manually SQL লিখতে হতো:

```sql id="dzj3hu"
INSERT INTO student ...
SELECT * FROM student ...
UPDATE student ...
DELETE FROM student ...
```

কিন্তু Spring Data JPA এগুলো automatically করে দেয়।

---

# এই Code টা দেখো

```java id="ztw7n7"
public interface StudentRepository
       extends JpaRepository<Student, Long> {

}
```

এখানে magic হচ্ছে:

```java id="wllnrb"
JpaRepository<Student, Long>
```

# Save Data

```java id="d3e5j0"

```

Equivalent SQL:


# Get All Data


Equivalent SQL:


